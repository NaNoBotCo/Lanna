Wat Phra Boromathat Doi Pha Som

วัดพระบรมธาตุดอยผาส้ม
compiled · unvisited

About

วัดพระบรมธาตุดอยผาส้ม เป็นวัดราษฎร์สังกัดคณะสงฆ์ฝ่ายธรรมยุติกนิกาย ตั้งอยู่ในตำบลแม่สาบ อำเภอสะเมิง จังหวัดเชียงใหม่ ปัจจุบันมีพระสรยุทธ ชยปัญโญ เป็นเจ้าอาวาส วัดพระบรมธาตุดอยผาส้มตั้งวัดเมื่อวันที่ 21 มิถุนายน พ.ศ. 2553 ได้รับพระราชทานวิสุงคามสีมาเมื่อวันที่ 18 เมษายน พ.ศ. 2555 วัดตั้งอยู่บริเวณดอยผาส้ม เป็นที่ประดิษฐานพระบรมสารีริกธาตุที่อยู่ในอำเภอสะเมิงมาช้านานแล้ว เมื่อครูบาศรีวิชัยได้จาริกมาที่สะเมิง ท่านได้ร่วมกับครูบาอุปาระ สร้างพระสถูปใหม่ ต่อมาเมื่อ พ.ศ. 2548 พระพุทธพจนวราภรณ์ เจ้าอาวาสวัดเจดีย์หลวงวรวิหาร จังหวัดเชียงใหม่ ได้นำคณะสงฆ์มาร่วมกันบูรณะพระสถูปนี้ โดยได้รับแรงศรัทธาจากชาวบ้านบ้านนอมลอง อังคาย และยั้งเมิน ได้สร้างพระมณฑปจตุรมุขครอบพระสถูป น้อมเกล้าฯ ถวายเป็นพุทธบูชา…
