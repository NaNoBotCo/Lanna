About

วัดเชตุพน เป็นวัดราษฎร์ สังกัดคณะสงฆ์ฝ่ายมหานิกาย ตั้งอยู่ในตำบลวัดเกต อำเภอเมืองเชียงใหม่ จังหวัดเชียงใหม่ ชื่อของวัดเชตุพนมาจากบริเวณนี้เป็นป่าทึบ เป็นซากวัดโบราณ จึงเรียกกันติดปากว่า "เจตะปน" ต่อมาได้ปรับให้เป็นภาษาไทยภาคกลางว่า "เชตุพน" วัดเชตุพนเป็นวัดร้าง ไม่ปรากฏว่าสร้างใดสมัยใดและใครเป็นผู้สร้าง ปรากฏซากวัดได้แก่ ใบเสมาและฐานอุโบสถ ฐานพระเจดีย์ และฐานพระวิหาร เป็นต้น จนในสมัยพระเจ้าอินทวิชยานนท์ เริ่มมีชาวบ้านมาตั้งถิ่นฐานบริเวณนี้จนกลายเป็นหมู่บ้าน ต่อมาราว พ.ศ. 2380 ได้มีพระภิกษุรูปหนึ่งชื่อ "คำ" เป็นผู้ทรงคุณวุฒิและมีศิลาจารวัตรปฏิบัติเคร่งครัดในพระธรรมวินัยมาก ได้มาจำพรรษาอยู่ที่วัดนี้ ญาติโยมมีความศรัทธาและเคา่รพนับถือมากจึงได้สร้างกุฏิ…
