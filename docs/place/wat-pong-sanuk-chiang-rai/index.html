Wat Pong Sanuk

★ registered Thai historic sitecompiled · unvisited

About

วัดปงสนุก เป็นวัดราษฎร์สังกัดคณะสงฆ์ฝ่ายมหานิกาย ตั้งอยู่ในตำบลเวียง อำเภอเชียงแสน จังหวัดเชียงราย มีที่ดินตั้งวัดเนื้อที่ 3 ไร่ 3 งาน 37 ตารางวา ตั้งอยู่ในเขตกำแพงเมืองด้านทิศตะวันออกติดกับริมแม่น้ำโขง วัดปงสนุก ไม่ปรากฏหลักฐานการสร้าง ชื่อวัดปงสนุกปรากฏในพงศาวดารภาคที่ 61 ที่กล่าวถึง ปงสนุกหรือปวงสนุก ได้รับการบูรณะก่อสร้างเมื่อ พ.ศ. 2040 โดยพ่อกำนันแดง สุปัชยา เป็นผู้นำพัฒนาวัด กรมการศาสนาระบุว่าตั้งวัดเมื่อ พ.ศ. 2040 ได้รับพระราชทานวิสุงคามสีมาเมื่อวันที่ 8 กุมภาพันธ์ พ.ศ. 2492 วัดนี้ยังเป็นชื่อที่เรียกโบราณสถานที่เขลางค์นคร เมืองเก่าจังหวัดลำปาง อาจแสดงถึงการติดต่อกันของชุมชนดั้งเดิม สันนิษฐานจากรูปแบบทางสถาปัตยกรรมได้ว่า น่าจะมีการก่อสร้างเมื่อประมาณพุทธศตวรรษที่ 21…
