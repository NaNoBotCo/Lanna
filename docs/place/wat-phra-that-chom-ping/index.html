Wat Phra That Chom Ping

วัดพระธาตุจอมปิง
compiled · unvisited

About

วัดพระธาตุจอมปิง เป็นวัดราษฎร์สังกัดคณะสงฆ์ฝ่ายมหานิกาย ตั้งอยู่ในตำบลนาแก้ว อำเภอเกาะคา จังหวัดลำปาง จากบันทึกเรื่องราวของชุมชนวัดพระธาตุจอมปิง ระบุว่าตำแหน่งที่สร้างพระธาตุจอมปิงนั้นเป็นวัดร้างที่มีมาตั้งแต่สมัยพระนางจามเทวี ต่อมาในสมัยพระเจ้าติโลกราชได้สร้างวัดขึ้นบริเวณซึ่งเป็นวัดร้างในพื้นที่ชุมชนของตน และตั้งชื่อว่า วัดจอมพี่เลี้ยง เพราะท่านนันทปัญญา พี่เลี้ยงของพระเจ้าติโลกราช ได้มาบูรณะวัดแห่งนี้ ใน พ.ศ. 2000 เกิดศึกพระยาใต้ยกทัพมาประชิดเมืองลำปาง เจ้าหมื่นด้งนครและพระเจ้าติโลกราชยกทัพมาช่วย และได้ฉลองชัยชนะที่วัดนี้ มีการตั้งชื่อวัดว่า วัดจอมพิงค์ชัยมงคล ต่อมาเรียกเพี้ยนเป็นวัดจอมปิงในปัจจุบัน จากข้อมูลของสำนักงานพระพุทธศาสนาแห่งชาติ ระบุว่าตั้งวัดเมื่อ พ.ศ. 2050…
