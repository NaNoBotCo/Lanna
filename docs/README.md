# Lanna Manuscript Wiki — static snapshot

A read-only, self-contained export of a working research catalogue of Lanna (Northern Thai) manuscripts and the living tradition around them.

This `docs/` folder is served by GitHub Pages. It is generated by `build_static.py` from the live wiki; edit the wiki, not these files.

High-resolution manuscript scans stay at their originating libraries; this snapshot links out to them. Concerns: 530kings@proton.me
