# Lanna Manuscript Wiki — [wichaa.net](https://wichaa.net)

A free, open archive of Northern Thai (Lanna) manuscripts and the living *wichaa*
(astrology, ritual, medicine, and magic) tradition around them — nearly 7,000
manuscripts, digitised primary textbooks, a curated knowledge graph, and the living
amulet market, on one continuous timeline. Every fact is recomputed from the catalogue,
not authored from priors.

**Live site → https://wichaa.net**
**Machine entry point → https://wichaa.net/llms.txt** (agents: read the JSON/RDF API, don't scrape the HTML)

This repository hosts the static build (`docs/`, served via GitHub Pages). The site is
generated from a local catalogue by a separate build pipeline — edit the generator, never
`docs/` directly.

## Support — free forever, funded by merit (*tam boon*)
The archive is free, with no ads and no paywall. The transcription and translation is done
by machines, and that machine-time costs a little — about **$0.06 per page**, a few dollars
for a whole manual. If it's useful to you and you're moved to, you can make merit: it buys
the bots time to read one more text so everyone can have it.

- ☕ **Ko-fi:** https://ko-fi.com/defiantchiangmai (message `TRANSLATE <manuscript#>` to sponsor a specific volume)
- 📖 **How support works:** https://wichaa.net/support — sponsored jobs run in public on the [activity feed](https://wichaa.net/activity)
- Specific manuscripts can be commissioned; the result still enters the free archive.
